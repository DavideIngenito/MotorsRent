# MotorsRent 🚗

MotorsRent è una piattaforma digitale per la gestione di una concessionaria e del noleggio auto. 
Il sistema è stato sviluppato come progetto per l'esame di Ingegneria del Software (Corso di Laurea in Informatica, Università degli Studi di Salerno).

## 🛠️ Tecnologie Utilizzate
Il progetto è stato sviluppato impiegando le seguenti tecnologie e strumenti:
* **Linguaggio di base**: Java
* **Sviluppo Web**: Jakarta EE, JSP (JavaServer Pages)
* **Database**: MySQL
* **Web/Application Server**: Tomcat / Glassfish
* **IDE & Versioning**: IntelliJ IDEA, NetBeans, Git

## 📋 Il mio contributo al progetto
Essendo un progetto sviluppato in team, mi sono occupato sia dell'implementazione tecnica che della stesura dell'architettura software. In particolare, ho curato:
* **Ingegneria dei Requisiti**: Redazione del *Statement of Work* (SoW) e del *Requirements Analysis Document* (RAD) per mappare le funzionalità core della piattaforma.
* **Design e Architettura**: Progettazione e stesura dell'*Object Design Document* (ODD).
* **Sviluppo Back-end**: Implementazione della logica di business e integrazione del database MySQL.
